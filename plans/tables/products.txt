Products Table (products) - done
| Column              | Type                    | Notes                                                 |
| ------------------- | ----------------------- | ----------------------------------------------------- |
| `id`                | BIGINT (PK)             | Primary key                                           |
| `name`              | VARCHAR(255)            | Product title                                         |
| `slug`              | VARCHAR(255)            | SEO-friendly URL                                      |
| `description`       | TEXT                    | Long description                                      |
| `short_description` | TEXT (nullable)         | For listing / preview                                 |
| `category_id`       | BIGINT (FK)             | Relation to `categories`                              |
| `brand_id`          | BIGINT (FK, nullable)   | If using brands                                       |
| `price`             | DECIMAL(10,2)           | Base price                                            |
| `sale_price`        | DECIMAL(10,2, nullable) | Discounted price                                      |
| `stock_quantity`    | INT                     | Inventory count (optional if you manage via variants) |
| `sku`               | VARCHAR(100)            | Stock keeping unit                                    |
| `is_active`         | BOOLEAN                 | Active / inactive                                     |
| `is_featured`       | BOOLEAN                 | For promotions / homepage                             |
| `meta_title`        | VARCHAR(255, nullable)  | SEO                                                   |
| `meta_description`  | VARCHAR(255, nullable)  | SEO                                                   |
| `created_at`        | TIMESTAMP               | —                                                     |
| `updated_at`        | TIMESTAMP               | —                                                     |
